## 任务一：Sidebar 收起态悬浮体验

涉及文件：
- client/src/components/Sidebar.tsx

改前：
- 侧栏收起后仅显示图标。
- 无 hover 延迟机制，用户在图标态难以辨识分类。
- 无毛玻璃浮层，无子分类预览。

改后：
- 新增 hover 延迟 200ms：使用 hoverTimer + onMouseEnter/onMouseLeave 防误触。
- 新增分类浮层：fixed 定位，z-index 9999，右侧弹出。
- 新增视觉：bg-white/80 + backdrop-blur-xl + 轻动画（framer-motion）。
- 新增子分类预览：若存在 children，在浮层中展示子项名称。
- 底部导航同样支持收起态 hover 浮层。

实现要点：
- 新增状态：hoveredCatId、hoveredLinkIdx、hoveredRect。
- 新增回调：handleCatMouseEnter/Leave、handleLinkMouseEnter/Leave。

## 任务三：ToolCard 高度强一致性

涉及文件：
- client/src/components/ToolCard.tsx
- client/src/components/ToolGrid.tsx

改前：
- 卡片高度受简介内容影响，网格可能错位。
- 描述区存在“展开/气泡”双模式，交互受滚动位置影响。

改后：
- 卡片固定高度：h-[200px]，并使用 flex flex-col 填充。
- 描述区固定为 line-clamp-2，超出省略。
- 悬浮补偿：300ms 后显示绝对定位浮层，展示完整描述，不撑开网格。
- 骨架屏同步固定高度：ToolCardSkeleton 也改为 h-[200px]。

实现要点：
- 删除旧 hoverMode 逻辑（expand/tooltip 双模式）。
- 新增 showTooltip + tooltipTimer（防抖展示）。

## 任务五：首页分区滚动 + Scroll Spy 联动

涉及文件：
- client/src/pages/Home.tsx
- client/src/components/Layout.tsx
- client/src/components/Sidebar.tsx

改前：
- 左侧分类点击主要是 URL 过滤 + 数据过滤。
- 右侧不是按一级分类 section 渲染。
- 没有滚动到可视中心后的侧栏高亮联动。

改后：
- 首页无搜索时改为 section 分区渲染：id=category-{id}。
- 左侧点击分类触发 scrollIntoView({ behavior: 'smooth' })。
- 使用 IntersectionObserver 监听主滚动容器并同步 activeSectionId。
- Sidebar 高亮逻辑接入 activeSectionId，实现滚动跟随高亮。

实现要点：
- Home 新增：toolsByCategory、scrollToCategory、IntersectionObserver effect。
- Layout 新增透传：activeSectionId、onScrollToCategory。
- Sidebar 新增逻辑：首页优先走滚动导航，不走过滤路由。

## 4) 风险与注意事项

1. CommandPalette 依赖浏览器环境和全局键盘事件，SSR 场景需额外守卫（当前项目是 Vite 前端，暂无阻塞）。
2. 批量“设为赞助/取消赞助”调用的是逐条 PUT，数量很大时请求次数较多。
3. Scroll Spy 在“有搜索词”时自动停用（设计使然，避免与过滤视图冲突）。
4. Sidebar 收起浮层使用 fixed + 高层级，若未来引入更高层弹窗系统需注意层级管理。
